CSS 고급 선택자

1) 연결 선택자

둘 이상의 선택자를 연결해서 스타일이 적용될 요소가 어느 부분인지 지정한다.

<연결 선택자>
종류 사용법 예시 설명
하위 선택자 (descendant selector) 상위요소 하위요소 section p 부모 요소에 포함된 하위 요소를 모두 선택한다.
자식 요소, 손자 요소, 손자의 손자 요소 등 모든 하위 요소에 적용된다.
section의 모든 하위(자식, 손자) p 요소에 스타일 적용
자식 선택자 (child selector) 부모요소 > 자식요소 section > p 부모 요소의 자식 요소만 선택한다.
자식 요소에만 적용된다 (손자 요소에는 적용안됨).
section 요소의 자식 요소인 p 요소에만 스타일 적용
인접 형제 선택자 (adjacent selector) 요소1 + 요소2 h1 + p 요소1과 요소2는 같은 레벨이면서 요소1 이후 가장 먼저오는 요소2를 선택한다.
h1의 형제인 요소 중 첫 번째 p 요소
형제 선택자 (sibling selector) 요소1 ~ 요소2 h1 ~ p 요소1과 형제인 요소2를 모두 선택한다.
h1의 형제인 요소 중 모든 p 요소

2) 속성 선택자

태그 안에서 사용하는 속성값에 따라 요소를 선택하는 역할을 한다. 속성 성택자는 속성값의 조건에 따라 특정 부분만 선택하기 쉬우므로 상황에 맞는 스타일을 지정하기 쉽다.

<속성 선택자 7가지>
선택자 (Selector) 예시 설명
[attribute] [lang] The attribute selector selects HTML elements with a given attribute set.
Selects all elements with a lang attribute.
[속성] a[href] 해당 속성이 있는 요소를 선택한다.
a 요소 중 href 속성이 있는 a 요소에만 스타일 적용
href 속성이 있으면 배경색을 aquamarine으로 지정한다
href없음
[속성 = 값] a[target = _blank] 지정한 속성값이 있는 요소를 선택한다.
a 요소 중 target 속성값이 _blank인 a 요소에만 스타일 적용
[속성 ~= 값] a[class ~= button] 지정한 속성값이 포함된 요소를 선택한다. (단어별)
class 속성값에 button이 포함된 요소에만 스타일 적용
속성값이 button인 요소에만 스타일 적용 (flat-button이나 buttons 등 정확하게 일치하지 않는 속성은 선택되지 않음)
[속성 |= 값] a[title |= us] 지정한 속성값이 포함된 요소를 선택한다. (단어별, 하이픈 포함)
title 속성값에 us가 포함된 요소에만 스타일 적용
title 속성값에서 속성값이 us이거나 us-로 시작하는 속성값을 선택한다.
[속성 ^= 값] a[title ^= eng] 지정한 속성값으로 시작하는 요소를 선택한다 (속성값이 정확하게 일치하지 않아도 지정한 속성값으로 시작하는 요소를 선택)
title 속성값에서 eng로 시작하는 속성값을 선택한다.
[속성 $= 값] a[href $= xls] 지정한 속성값으로 끝나는 요소를 선택한다.
href 속성값에서 xls로 끝나는 a 요소에만 스타일 적용
(href에 링크된 파일의 확장자가 xls인 요소를 찾는다)
[속성 *= 값] a[href *= w3] 지정한 속성값의 일부가 일치하는 요소
href 속성값 중에 w3가 포함된 요소에만 스타일 적용
