1) :root 가상 클래스와 var() 함수

변수 만들기

  • :root 가상 클래스는 CSS 변수를 선언하기 위한 용도로 사용하고, CSS 코드의 다른 선택자보다 위에 작성한다.
  • CSS 변수 이름 앞에는 반드시 하이픈 2개를 붙여야 한다.
  • 변수 이름은 대소문자를 구분한다.
  • 변수 이름에는 문자와 숫자, 하이픈, 언더바를 포함할 수 있다.
  • 참고. :root가 아닌 다른 요소에서도 CSS 변수를 선언할 수 있지만, 이 경우에는 해당 요소와 그 하위 요소에서만 사용할 수 있다.

:root {
  --bg-color: #222;
}

변수 사용하기

  • var() 함수는 CSS 변수 값을 가져올 때 사용합니다. var()함수 괄호 안에 변수 이름을 넣으면 됩니다.

:root {
  --main-bg-color: #222;
  --main-text-color: #fff;
}
body {
  background-color: var(--main-bg-color);
  color: var(--main-text-color);
}

2) 계산식으로 값을 결정하는 calc() 함수

  • 괄호 안에는 크깃값이나 각도, 시간, 백분율, 숫자 등 다양한 단위의 값을 사용한 계산식이 들어간다.
  • 계산식에는 4칙연산 연산자(+, -, *, /)도 사용할 수 있다.
  • +, - 사용 시에는 연산자 앞뒤로 공백을 두어서 다른 기호와 구별될 수 있도록 한다.
  • 곱하기 연산에서는 2개의 피연산자가 모두 숫자여야 하고, 나누기는 오른쪽 피연산자가 0이 아닌 숫자여야 한다.

nav 요소의 너비를 전체 너비에서 80px만큼 뺀 값으로 설정

nav {
  width: calc(100% - 80px);
}

3) 최소, 최대값을 적용하는 min(), max() 함수

  • min() 함수는 괄호 안에 나열된 값 중에서 가장 작은 값을 반환한다.
  • max() 함수는 괄호 안에 나열된 값 중에서 가장 큰 값을 반환한다.

.heading 요소의 너비를 100%와 700px 중 작은 값으로 적용한다.

.heading {
  width: min(100%, 700px);
}

.heading h1 요소의 폰트 크기를 3vw, 2em 중 큰 값으로 적용한다.

.heading h1 {
  font-size: max(3vw, 2em);
}

4) 값의 범위를 제한하는 clamp() 함수

  • clamp(최소값, 최적값, 최대값)

.heading h1 요소의 폰트 크기를 2vw로 지정한다. 최소 1rem보다 작을 수 없고 최대 2rem 보다 커질 수 없다.

.heading h1 {
  font-size: clamp(1rem, 2vw, 2rem);
}

5) 다양한 시각 효과를 추가하는 filter 속성과 함수

  • filter: 함수
  • 필터 함수를 2개 이상 사용할 때는 공백으로 구분한다.
  • filter: blur(radius)
    • blur(8px); /* blur with 8px radius */
    • blur(1.17rem); /* blur with 1.17rem radius */

    • .blur { filter: blur(5px); }

  • filter: brightness(amount)
    • amount는 숫자나 %로 입력한다.
    • brightness(1); /* Brightness of input is not chagned */
    • brightness(0); /* Brightness is reduced to zero, so input turns black */
    • brightness(1.5); /* 1.5배 밝아진다. 150%로 입력해도 된다. */

    • .brightness { filter: brightness(80%); }
